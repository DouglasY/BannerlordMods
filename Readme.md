## Mod收集

### 项目内容
> 包含可直接使用的Mod，以及Mod源码。
> 尽可能的收集一些Mod，当然前提是得到原作者的同意。

### 目录结构
```shell
BannerlordMods # 主目录
├── ExampleMod # 单个Mod目录
│   ├── Mod    # Mod的内容
│   │   └── ExampleMod
│   │       ├── SubModule.xml
│   │       └── bin
│   │           └── Win64_Shipping_Client
│   │               └── ExampleMod.dll
│   └── Source # Mod的c\#源码
│       ├── ExampleMod
│       │   ├── Class1.cs
│       │   ├── ExampleMod.csproj
│       │   └── Properties
│       │       └── AssemblyInfo.cs
│       └── ExampleMod.sln
├── LICENSE
└── Readme.md
```
